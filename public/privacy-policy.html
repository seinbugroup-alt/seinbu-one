1. Données collectées

SEINBU ONE collecte uniquement les données fournies par Pi Network lors de l'authentification : identifiant unique Pi (UID) et nom d'utilisateur Pi. Aucun mot de passe n'est stocké.

Les données sont utilisées exclusivement pour personnaliser l'expérience utilisateur dans les 12 modules de SEINBU ONE et pour traiter les paiements Pi Network.

3. Partage des données

SEINBU GROUP SA ne vend ni ne partage vos données personnelles avec des tiers, sauf obligation légale.

4. Sécurité

Toutes les communications sont chiffrées via HTTPS. Les paiements Pi sont traités exclusivement via le SDK officiel Pi Network.
